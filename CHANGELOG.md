# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-27

### Added

- Initial public baseline of MetaMemory Studio.
- Full backend API for ingest, runs, memory operations, policy evolve, drift, and export.
- Frontend Next.js experience with runs timeline, contribution compare, drift, memory library, and themes.
- Hash-chain integrity verification, PII detection/redaction, and memory governance flows.
- Unit, integration, and Playwright E2E test suites.
- CI workflow covering backend checks, frontend checks, and E2E.
